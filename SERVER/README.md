### This is the Backend part of the `KNOWLEDGEB-BLOG` website

#### Used Tech:

-   Express JS
-   Cors
-   Dotenv
-   mongoose
-   jsonwebtoken
-   nodemon
-   http-errors
-   Multer

#### Routes:

-   Base URL: http://localhost:2000/
-   Add New User: http://localhost:2000/api/v1/people/add-people
-   Get All Users: http://localhost:2000/api/v1/people/get-people
